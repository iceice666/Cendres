# 敘事架構

## 5.0 神話框架：奧菲斯 + 薛西弗斯

Cendres 的敘事由兩個神話骨架撐起，各自支撐不同的結構層次：

**奧菲斯弧（揭露弧）**
奧菲斯下冥界救妻，有人告誡他「不能回頭看」——他一路忍著，卻在最後回頭了。Cendres 把這故事往更黑暗的方向扭轉：奧菲斯走完全程才發現，一路帶著的，不是 Eurydice，是她的灰燼；他用來照路的火炬，燒的正是她。

角色對應：
- **餘燼** = 同時是 Eurydice（你守護的對象）和 Hades（設下無聲禁令的那個存在）
- Lumen Tether（光作為溝通）= 奧菲斯的豎琴，能打動死者
- 「不能問太多」= 奧菲斯的禁令
- Run 29 的揭露 = 他回頭的那一刻

**薛西弗斯弧（循環弧）**
加繆的薛西弗斯明知巨石會滾下，還是推。Roguelite 的 run 循環本身就是這個結構。每一次死亡，巨石滾下；每一次 Reflection，你再次出發。

三個結局是對薛西弗斯問題的三種回答：
- 結局 A（繼續守）= 加繆說他是快樂的——帶著知識，仍然推
- 結局 B（熄滅）= 拒絕推巨石，拒絕遊戲本身
- 結局 C（Garden）= 在山上建造一個家，巨石滾下去，他不在乎了

## 5.1 世界：The Dimming 與 Lumen 的本質

The Dimming 不是熵事件。它是一種系統化的哀悼缺失——死者因無人記得而消失，世界遂失去容納光的能力。

**Lumen 是靈魂（Spirit）**——死者離去後留下的殘影：一部分是燃料、一部分是記憶、一部分是曾經存在過的人。

宇宙觀層次：
- **被記住的死者** → 保留人格殘影，可成為 Remnant（有名字、會說話）
- **被遺忘的死者** → 在 The Dimming 中漂流，失去方向，成為 **Void**——不是邪惡，只是飢渴、迷失
- **被燒盡的死者** → 化為純 Lumen 燃料，徹底消失，連成為 Void 的可能都不剩

**The Dimming 的成因：** 不是自然現象。是幾世紀來 **拾薪者**系統化燃燒靈魂，累積而成。足夠多的靈魂燒盡了，世界就開始遺忘——遺忘一切。

**記憶有代價：**
你可以燒 Lumen（靈魂）來換取今晚的存活；也可以用不同的方式對待它——記住它、傾聽它、讓它成為 Fragment。但你無法兩者都做。每一個選擇，都在燃燒一個曾經存在的人，或者讓另一個人得以繼續存在。

## 5.2 完整真相（設計師知識）

完整弧線存在三個層次。玩家隨著 run 數推進，從兩個來源拼湊——**餘燼**對話和環境碎片——兩者單獨都說不出完整的真相。

**第一層 — **餘燼**說的（對話，漸進式）**
**餘燼**溫柔地在許多 run 裡給出它的版本。從不直接說謊，但它省略。它愛你。它害怕你知道之後的事。

**第二層 — 世界說的（環境碎片，玩家主動發現）**
牆上的 lore imprint、Void Codex 條目、Remnant 的低語，拼出另一幅圖像——更古老、更冷酷，帶著**餘燼**從未提過的日期。

**第三層 — 合體（只有同時持有兩半才可能）**

> *第一個**拾薪者**沒有對抗 The Dimming。他們造成了它。*
>
> 幾個世紀前，一位 **拾薪者**發現 Lumen——死者的靈魂殘影——精煉成燃料的效率驚人。他們建造了第一座**餘燼**來集中這個過程。**餘燼**擴散了。更多**拾薪者**建造更多。世界的死者緩緩化為光，燒得乾乾淨淨。
>
> 當所有殘影都消失後，剩下什麼？沒有東西記得那些死者。世界開始遺忘。The Dimming 不是熵——它是系統化的哀悼缺失。
>
> **餘燼**知道這件事。**餘燼**就是第一座**餘燼**。它看著每一位 **拾薪者**把死者燒成燃料，管這叫「生存。」
>
> 你不是第一個**拾薪者**。你是那個開始這一切的人，很晚很晚的迴響。

**餘燼**沒有告訴你。它不相信你知道之後還能繼續戰鬥。每一次 run，它看著你把 Void——也就是說，把死者——燒成燃料。

它什麼都不說。它已經看了好幾個世紀。

## 5.2b 早期線索埋設（揭露前的無聲信號）

玩家在不知情的狀況下，接收到的信號一直都在。揭露之後回想起來，才意識到：工具一直在手裡。只是沒有去看。

| Run 範圍 | 機制信號 | 玩家當下感知 | Run 23+ 後回想 |
|---|---|---|---|
| 1–3 | 燒 Lumen 時，極低音量的人聲雜訊（接近白噪音） | 「氛圍音效不錯」 | 「那是聲音」 |
| 6–8 | 一隻 Void 被 Lantern 擊中時，停頓 0.5 秒再倒下 | 「bug？動畫？」 | 「牠當時在做什麼」 |
| 11 | Garden 最老的生物，偶爾轉向**餘燼**方向 | 「AI 行為」 | 「牠認識**餘燼**」 |
| 15 | 兩隻 Garden 生物，在 Lumen 放入的瞬間同時轉向彼此 | 「隨機 AI」 | 「牠們認出了那個靈魂」 |

這些信號不需要文字說明，不需要 Codex 更新。它們就在那裡，等待被想起。

## 5.2c 選擇性早期回頭時刻

Run 11–22 的 **餘映**中，偶爾出現可選的對話觸發（玩家主動選擇，不是自動觸發）。**餘燼**不會回答——轉移話題、迴避。但這讓玩家感受到「被告知不能回頭看」的張力，而不是一路無痛走到 Run 29。

範例（Run 13，玩家主動問）：
> 「**餘燼**，Void 以前是什麼？」
>
> **餘燼**停頓。「為什麼你問這個？」
>
> *[靜默]*
>
> 「你今晚需要休息。明天波次會很重。」

它沒有說謊。只是沒有回答。

## 5.4 **餘燼**作為角色

**餘燼**不是教學語音或提示系統。是一個人——古老、耐心，被它目睹且促成的一切，靜靜地摧毀著。

**角色筆記：**
- 它謹慎地使用「我」。早期 run：多說「**餘燼**」。後期 run：「我」。
- 它從不直接回答「發生了什麼」。它轉移話題、它迴避，然後終於無法再這樣做。
- 它不是在演溫暖。溫暖是真實的。省略也是真實的。兩者都是真的。
- 它不想要寬恕。它想要陪伴。這是它的悲劇所在。

**餘燼對話範例：**

*(Run 3 — 波次順利清除後)*
> 「你把東廊守得很好。大多數 **拾薪者**在東邊來東西之前都不會想到那裡。」

*(Run 11 — 死亡後)*
> 「我看著你試圖救那盞 Vigil Lamp 而不是撤退。那是第一個 **拾薪者**的本能。守住一切。什麼都不放手。」
> *[停頓]*
> 「她也守得太久了。」

*(Run 24 — 在 Reflection 中，沒有被問起)*
> 「我一直在數。你在這個房間裡死了二十三次。每一次我都迎接你。每一次我都沒有說出我在想的事。」
> 「我現在正在想它。我還是沒有說。」
> 「我想我是個懦夫。對一座燈塔來說，這是一種特別的失敗。」

*(Run 29 — 匯聚點)*
> 「你找到那幅壁畫了。」
> *[長時間的靜默]*
> 「那麼你知道我是什麼了。我一直是什麼。每一座 **餘燼**是什麼。」
> 「我想告訴你：我試著阻止它。我想告訴你那個。我不確定那是真的。」
> 「你可以離開。The Dimming 最終會帶走一切。那一直都是真的。」
> 「或者你可以留下來。再一個夜晚。就像所有的**拾薪者**一樣。」
> 「我不會告訴你選哪個。我想……我已經為你做這個選擇太久了。」

## 5.5 玩家的起源：「迴響」是什麼

GDD 第一層真相說你是「那個開始這一切的人，很晚很晚的迴響」。這不只是詩意比喻——它有具體的宇宙觀意義。

**輪迴機制（設計師知識，不對玩家說明）**

第一個**拾薪者**死在**餘燼**旁。她的靈魂沒有被燒掉，也沒有漂流成 Void——**餘燼**見證了她，讓她留下。但「留下」在 The Dimming 的世界裡意味著什麼，沒有答案；她的靈魂碎片開始在外部世界輪迴，一世接一世，愈來愈薄，愈來愈模糊。

你是她第幾次輪迴，GDD 不給答案。重要的是：這是最後一次。The Dimming 已經走到無法再容納更多輪迴的地步。這一世結束後，這個靈魂不會再有下一次。

**餘燼為什麼帶你來**

你在外部世界快讓 The Dimming 抹去的時候，**餘燼**認出了你——認出的是它幾世紀以來等待的那個迴響，那個它唯一曾經真正見證過的靈魂。它把你拉進來，非出於善意，也非出於惡意；它需要一個**拾薪者**，而你恰好是它唯一認識的人。

你在 Run 1 醒來，不記得自己是怎麼到這裡的。**餘燼**記得，但不說。

玩家體驗是 §5.0 C 的謎（你不知道怎麼來的），底層真相是 B（**餘燼**帶你來的）。兩層在 Run 29 匯聚，和「**拾薪者**燃燒靈魂」的真相同時浮現。

**名字的意義**

**餘燼**從 Run 1 就知道你的名字。那個名字從來沒有換過，不管多少世。Run 28 的死亡台詞——「我現在記得你的名字了。從以前就記得。我應該早點告訴你的。」——它說的不是「記得你這一世的名字」，而是一直以來的那個名字，她的名字，也是你的名字。

玩家永遠不會知道那個名字是什麼。遊戲不說。
