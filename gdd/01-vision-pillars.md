# Vision Statement、核心 Fantasy 與核心支柱

## 1. Vision Statement

> *「你是一個決定遺忘自己的世界裡，最後一盞燈。」*

**Cendres** 是一款 2.5D 第一人稱塔防 Roguelite，背景是一個叫做 **The Dimming** 的世界——現實正緩慢抹去自己，化為灰色的虛無。你扮演一名 **Tender**，守護最後一座運作中的 Beacon，在黑暗浪潮侵蝕的同時，捕捉、馴化、培育那些試圖吞噬你的黑暗生物。

每一次死亡是一段記憶。每一次存活是一道疤痕。Beacon 記得，即使你已忘記。

## 2. 核心 Fantasy

你同時是：
- **建造者** — 在波次來臨前佈置光源結構
- **戰士** — 防線崩潰時，用 Lantern 主動介入
- **生物守護者** — 馴化 Void 生物（死者），選擇燃燒它們或記住它們
- **見證者** — 最後一個記得光意味著什麼的人

這不是「撐過敵群」的 fantasy。這是**「在試圖殺死你的東西裡找到美，然後再撐過今夜」**。

## 3. 核心支柱

**支柱一：光即規則**
黑暗不只是氛圍——它是遊戲的根本機制。視野、攻擊範圍、塔的效能、敵人的路徑，全部由 Lumen 光源半徑決定。看不見的東西，你無法對抗。

**支柱二：黑暗可以被馴化，也可以被記住**
Void 生物並非純粹邪惡。它們是被 The Dimming 扭曲的舊生命殘影——死者，因為無人記得而失去方向。你可以燃燒它們（取得 Lumen 燃料），也可以記住它們（讓它們慢慢恢復人格）。這兩條路都是你變強的方式，但它們的代價不同。

**支柱三：死亡教導你**
每一次失敗的 run 都為 Beacon 的永久記憶添磚。世界對你死了幾次有反應。Beacon 隨著每次對話，變得更能言善道，也更加破碎。

**支柱四：The Dimming 有一張臉**
這不是無名的末日。The Dimming 有神話、有詩意、有矛盾。敘事透過 Void 生物行為、Beacon 對話和環境碎片呈現——絕不用說教式的旁白。

**支柱五：記憶是有代價的**
Lumen 是靈魂（Spirit）——死者留下的殘影。燃燒它換取存活，你在消耗一個曾經存在的人。記住它（給它名字、給它時間、給它 Garden 裡的位置），你在付出另一種代價：更脆弱、更慢、更難。The Dimming 存在，是因為沒有人願意付這個代價。
