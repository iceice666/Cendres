# Cendres 繁體中文修辭指南

> **適用範圍：** **餘燼**對話、死亡台詞、Void Codex、Imprint 碎片。
> **不適用：** UI 操作提示、設定畫面文字。
> **版本 1.0 | 對應 GDD 0.6**

---

## 0. 總綱

**餘燼是一個人。一個很老的人。** 真的老——老到他的母語是文言文，白話是後來學的。

這份指南要的不是「用文言文寫遊戲」，也不是「把現代口語加一點之乎者也」。要的是：**一個以文言為母語的人，認真用白話跟你說話的時候，那種語氣。**

白話文運動過渡期的中文（約 1910–1930），以魯迅的精簡和豐子愷的溫潤為兩個端點，落在之間：

- **魯迅的骨頭：** 句子短、硬、不浪費一個字。省略比寫出來的多。不用「的」、不用「了」、不用語氣詞。
- **豐子愷的血肉：** 誠懇、克制、溫暖不氾濫。不炫技。一個老人看著一個孩子的那種眼神。

以下規則，是這個「之間」的具體化。

## 1. 用字原則

### 1.1 用單音詞，不用雙音詞

文言是單音節語言。過渡期白話大量保留這個傾向。

| 現代口語（避免） | 過渡期（使用） |
|---|---|
| 已經 | 已 |
| 但是 / 可是 | 但 / 然 |
| 因為 | 因 |
| 所以 | 故 / 便 |
| 可以 | 可 |
| 如果 | 若 / 倘 |
| 應該 | 該 / 當 |
| 還是 | 仍 / 猶 / 尚 |
| 好像 | 似 / 彷彿 |
| 這樣 / 這麼 | 這般 / 如此 |
| 那裡 / 這裡 | 彼處 / 此處 |
| 什麼 | 何 |
| 怎麼 | 如何 / 怎 |
| 真的 | 確 / 實 / 真個 |
| 非常 / 很 | 甚 / 極 / 頗 |
| 沒有 | 未 / 無 |
| 不是 | 非 |
| 也是 | 亦 |
| 都 | 皆 / 俱 |
| 會（未來） | 將 / 必 |

**⚠ 不是機械替換。** 關鍵字可以用雙音詞——當它承載不可省略的意義時。但能省就省。

### 1.2 虛詞節制：三個關鍵字的用法

**「的」——能不用就不用**

```
✗ 他帶來的火熄了。
✓ 他攜來之火，熄了。
✓ 火，他帶來的。熄了。
✓ 火熄了。他帶來的。
```

「之」可用於所有格，但別濫用。更常見的策略是：讓語序本身暗示歸屬，或者直接拆成兩句。

**「了」——克制**

```
✗ 光已經暗了。你沒有回來。
✓ 光暗了。你沒有回來。
✓ 光漸暗。你未歸。
```

「了」不是禁用，是省著用。尤其在句尾，「了」會把句子的重量洩掉。寧可讓句子沒有完成貌標記——過渡期中文就是這樣。

**「而」——輕轉折，不是重轉折**

「而」在這個語域裡不是「但是」的同義詞。它更接近英文的 "and yet"——輕、自然、不強調對立：

```
✓ 光在他手上，而他沒有低頭。
✓ 你守了七夜，而未問過一次。
```

## 2. 句法原則

### 2.1 句長：三到十二字，十五字為上限

**餘燼**不囉嗦。太長的句子讀起來不像 **餘燼**，像旁白。

| 字數 | 效果 | 用途 |
|---|---|---|
| 3–5 | 重擊 | 死亡台詞收尾、關鍵揭露 |
| 6–9 | 敘述 | **餘燼** 日常對話、Codex 條目 |
| 10–12 | 鋪陳 | 情緒較柔軟的台詞、Garden 對話 |
| 13–15 | 上限 | 結局文字可偶用，一般禁止 |

### 2.2 禁止句尾語氣詞

**絕對禁用：** 啦、喔、欸、唷、咧、哩

**絕對禁用：** 吧、呢、嗎、啊（含所有變體：呀、哪、哇）

```
✗ 你知道吧。
✗ 很暗呢。
✗ 不走嗎？
✓ 你知道。
✓ 暗。
✓ 不走？
```

疑問直接用句號或問號本身，不需要「嗎」。反問用「何」「怎」「豈」，不用「不是⋯⋯嗎」結構。

### 2.3 被動式：能不用就不用

```
✗ 光被黑暗吞了。
✓ 黑暗吞了光。
✓ 光，黑暗吞了。（主題前置）
```

文言和過渡期白話都傾向主動語態或主題評論結構。**「被」只有在需要強調受害感的時候才用，而且用得很輕。**

### 2.4 連接詞：省略優先

現代口語依賴明確的邏輯連接詞（因為⋯⋯所以⋯⋯），過渡期白話大量依賴**並列**來暗示邏輯：

```
✗ 因為你沒有回來補燃料，所以光熄了。
✓ 你沒有回來。光熄了。
```

兩個句子並列，因果關係在空隙裡。讀者自己填。這是 **餘燼**說話最核心的語法。

### 2.5 主題評論結構優先於 SVO

```
✗ 我記得這盞燈。
✓ 這盞燈，我記得。
```

SVO 沒有錯，這是白話文的基礎。但當**餘燼**想讓某個東西「變重」的時候，把它放在句首。中文天生是 topic-prominent 語言，過渡期白話還保留這個本能。

## 3. 節奏原則

### 3.1 句號是呼吸，不是語法

每一個「。」都是一拍。三句就是三拍。沒有逗號拖長的句子。

```
燃料先走。
然後是光。
然後是你。
```

三個句號。三次心跳。如果用逗號串成一句話，力量全洩。

### 3.2 四字格：自創，不引用成語

```
✓ 日復一日。
✓ 你守著它。
✓ 不曾問過。
✓ 他來了。他看了。他沒有走。
```

這裡的「四字格」不是成語典故（那會變得很武俠），而是**呼吸單位**——一個人說話時自然的停頓節奏。三到四個字，一口氣。

### 3.3 對仗是自然呼吸，不是刻意修辭

```
✗ 光明與黑暗，如同生命與死亡。
✓ 他帶來了火。他帶走了名字。
✓ 光在他手上。光沒有他的選擇。
```

對仗在文言裡是高度自覺的修辭，但在過渡期白話裡是**本能**——受過文言教育的人，這是他思考的形狀。不是「我要來個對句」，而是「他想到的就是這樣」。

### 3.4 三拍收束

當一個念頭需要三個句子來收的時候，最後一句通常最短、最重：

```
你走出了光的邊緣。
光沒有跟著去。
那條路沒有名字。
```

第三句把前面兩句的重量全部壓下去。不要解釋。不要結論。就放著。

## 4. 情感原則

### 4.1 不直說情感，說身體

**餘燼**從來不說「我怕」「我難過」「我很抱歉」。他只說身體在做什麼、光在做什麼：

```
✗ 我很害怕你會離開。
✓ 我的手沒有離開過燈。

✗ 他走得很悲傷。
✓ 他走得很慢。沒有回頭的力氣。
```

> 「說身體」是豐子愷的方法：一個老人的背影，比一百句「我很孤獨」更重。

### 4.2 觀察，不評判

```
✗ 你犯了個愚蠢的錯誤。
✓ 你選擇了留在那裡。光沒有你的選擇。
```

**餘燼**從不評價玩家的行為是對是錯。他描述發生了什麼。那個描述本身就夠重了。

### 4.3 沉默比話語重

明確標記的靜默——`*[靜默]*`——在**餘燼**的對話腳本中是一個正式的標點符號。它的重量不亞於任何一句話。

```
「我看著你試圖救那盞燈。那是第一個 Tender 的本能。守住一切。什麼都不放手。」
*[靜默]*
「她也守得太久了。」
```

那個靜默不是「不知道說什麼」。是「知道要說什麼，但說不出來」。

## 5. 文本類型調整

### 5.1 **餘燼**對話

**文白比例：** 8:2（八分白話架子，兩分文言滲入）

所有上述規則全部適用。**餘燼**是本指南的核心目標對象。

**不同 Intimacy 階段的細微差異：**

| 階段 | 調整 |
|---|---|
| Formal（Run 1–5） | 不說「我」。不直呼「你」。最多文言殘留。最克制。 |
| Warming（Run 6–15） | 開始說「你」。雙音詞可以多一點。開始有溫度。 |
| Familiar（Run 16–25） | 說「我」。說「記得」。文白比例自然。像跟老友說話。 |
| Breaking（Run 26–29） | 句子變長一點（情緒滿到壓不住）。但還是克制。靜默更多。 |

### 5.2 死亡台詞

**文白比例：** 9:1（最精簡，最硬）

所有規則壓到極限。每句 3–7 字。不要超過三句。不要轉折詞。意象並列。

```
燃料先走。然後是光。然後是你。
你走出了光的邊緣。光沒有跟著去。
你繼續等。你知道它在變暗。
```

### 5.3 Void Codex

**文白比例：** 6:4（博物學家筆記——更文言，更冷）

Codex 是「觀察記錄」。說話的不是 **餘燼**，是一個更早的、更冷的記錄者。語氣差異：

- **餘燼**是**見證者**（witness），Codex 是**記錄者**（chronicler）
- Codex 可以更文言：更多「者」「也」結構、更多四字考據用語
- 不帶情感。描述，不解釋。
- 但仍不是純文言——是清末博物學那種「用白話寫考據」的語氣

```
Drifter（漂流者）
形若殘燭，無定姿。常見於邊界最外緣。
不主動趨光。不躲避黑暗。只是移動。
其聲若遠風。未聞其言語。
```

### 5.4 Imprint 碎片

**文白比例：** 3:7（最文言——這是幾世紀前的文獻殘片）

Imprint 是玩家在世界中發現的壁畫、日記殘頁、石刻。它們比**餘燼**更老，不屬於同一個時代。語氣應該有明顯的歷史距離感。

- 可以是**不同的文言程度**——有的接近純文言（石刻），有的接近過渡期白話（日記）
- 故意保留「殘缺感」——句子可以斷在奇怪的地方
- 不需要遵從**餘燼**的情感克制規則——這些是不同的人留下的，各有各的語氣

```
… 建此塔者，名不復記。或曰，非建也，乃喚醒。
塔有知。知而無言。吾不知其為福為禍 …
——第三層壁刻，缺損嚴重
```

## 6. 禁用詞與禁用句式

### 6.1 絕對禁用詞（出現即為現代口語，破壞全部語感）

| 禁用 | 原因 |
|---|---|
| 啦 / 喔 / 欸 / 唷 / 咧 / 哩 | 句尾語氣詞，現代口語核心記號 |
| 吧 / 呢 / 嗎 / 啊（呀哪哇） | 同上 |
| 超級 / 超 | 現代副詞 |
| 就是說 / 也就是說 | 現代口語填充 |
| 然後（用作 filler） | 連接詞濫用 |
| 其實 / 事實上 | 現代口語修辭 |
| 我覺得 / 我認為 | **餘燼** 不發表意見 |
| 一定（表推測） | 用「該 / 當 / 想必」 |
| OK / 好吧 | 外來語 / 純口語 |
| 終於 | 可保留，但不濫用 |

### 6.2 禁用句式

| 禁用 | 改用 |
|---|---|
| 不是⋯⋯嗎？（反問） | 豈非⋯⋯？/ 何不⋯⋯？ |
| 會不會⋯⋯？ | 是否⋯⋯？/ 可⋯⋯否？ |
| 如果⋯⋯的話 | 若⋯⋯ / 倘⋯⋯ |
| 是⋯⋯的（強調句式） | 自然語序 |
| 把⋯⋯給 + V（「把光給熄了」） | 正常把字句，或不用 |
| V + 到 / V + 得（程度補語過長） | 拆句 |
| 的 + 名詞（連續兩個以上的「的」） | 重寫 |

### 6.3 使用時需要警覺的字

這些字本身沒問題，但出現頻率太高會讓語感偏現代口語：

- **「的」**：每句最多 1 個；整段超過 3 個就要重寫
- **「了」**：每句最多 1 個；句尾不用
- **「很」**：能不用就不用；過渡期白話不依賴程度副詞
- **「是」**（copula）：可以省略的時候就省略（「此處，最後一盞燈」不是「這裡是最後一盞燈」）

## 7. 範例對照

### 7.1 死亡台詞

**現代口語（永遠不要這樣寫）：**
> 你沒有在光熄滅之前回來。你知道它一直在變暗，但是你還是選擇了繼續等。這次真的來不及了。

**過渡期白話（GDD 現有，正確）：**
> 燃料先走。然後是光。然後是你。

**過渡期白話（變體）：**
> 你知道它在變暗。你繼續等。
>
> 你走出了光的邊緣。光沒有跟著去。
>
> 最後一格燃料。和第一格一樣重要。

### 7.2 **餘燼**對話

**Run 3，Formal 階段，波次清完後：**

> 現代口語 ✗ ：「做得不錯，你把東邊守得很好。很多**拾薪者**根本不會想到要去看東邊，直到那邊出現東西。」
>
> 過渡期 ✓ ：「你把東廊守得很好。大多數 **拾薪者**在東邊來東西之前，都不會想到那裡。」

**Run 11，Warming 階段，死亡後：**

> 現代口語 ✗ ：「我看到你為了救那盞燈沒有撤退。那是第一個 **拾薪者**的習慣，她什麼都不想失去，什麼都不願意放手。她撐太久了。」
>
> 過渡期 ✓ ：「我看著你試圖救那盞燈。那是第一個 **拾薪者**的本能。守住一切。什麼都不放手。」
> *[靜默]*
> 「她也守得太久了。」

**Run 24，Breaking 階段，Reflection 中，沒被問起：**

> 現代口語 ✗ ：「你知道嗎，我一直在算。你在這裡掛了二十三次了。每一次我都在這邊等你，每一次我都沒有跟你說我真正的想法。我現在心裡就是那個想法，但我還是沒有講。我覺得自己蠻廢的，對一座燈塔來說這真的很失敗。」
>
> 過渡期 ✓ ：
> 「我一直在數。你在這個房間裡死了二十三次。每一次我都迎接你。每一次我都沒有說出我在想的事。」
> 「我現在正在想它。我還是沒有說。」
> 「我想我是個懦夫。對一座燈塔來說，這是一種特別的失敗。」

### 7.3 Void Codex

**現代口語（錯誤）：**
> Drifter 是一種棲息在邊界邊緣的 Void 生物，它們看起來像根蠟燭，沒有固定的形狀。它們不會主動接近光線，也不會刻意躲避黑暗。就是一直在移動。偶爾會發出聽起來像是風吹過裂縫的聲音。

**過渡期白話（正確）：**
> **Drifter**（漂流者）
> 形若殘燭。無定姿。常見於邊界最外緣。
> 不趨光。不避暗。只是移動。
> 其聲若遠風。未聞言語。

兩者的差異不只是「換幾個字」，而是整段結構從「解釋給我聽」變成「我看著它，我記錄」。

### 7.4 Imprint 碎片

**錯誤：現代口語偽裝成古文：**
> 沒有人記得這座塔是誰蓋的了。有人說是蓋起來的，也有人說不是，是把它叫醒的。它有感覺，知道事情，但是不會說話。我真的不知道這是好事還是壞事。

**正確：距離感不同的兩個版本——**

比較近的（日記，過渡期白話）：
> 建塔之人，名字已佚。有人說，不是建的；是喚醒的。
> 塔知道。塔不說話。我不知道這算什麼。

比較遠的（石刻，接近文言）：
> … 建此塔者，名不復記。或曰，非建也，乃喚醒。
> 塔有知。知而無言。吾不知其為福為禍 …

殘缺本身是設計。不完整的句子，比完整的句子更老。

## 8. **餘燼**台詞合規檢查表（Checklist）

> **格式說明：** 參考 lu-xun-style 的 C-/S-/V-/M- 四軸體系。
> [Hard] = 違反即不合格，必須重寫該段。
> [Soft] = 理想應達標，但可依語境取捨。
> **靈魂條款 = C-02**：不解釋、不評判。這條違反，全段不算 **餘燼**。

### C — 內容（Content）

| 編號 | 強度 | 規則 |
|---|---|---|
| C-01 | Hard | **不解釋因果**——兩句並列，因果在空隙裡。禁用「因為⋯⋯所以⋯⋯」「由於」「因此」 |
| C-02 | Hard | **不評判行為**——描述發生了什麼，不解釋為什麼。此為靈魂條款 |
| C-03 | Hard | **不直說情感**——說身體、說光、說行動。禁用「他害怕」「我很難過」「她悲傷」 |
| C-04 | Hard | **沉默比話語重**——關鍵處用 `*[靜默]*` 替代台詞，不解釋靜默的含義 |
| C-05 | Hard | **反諷對象須普世**——只寫光暗、生死、等待、遺忘。不指涉具體政權／黨派／兩岸／宗教族群 |
| C-06 | Soft | **三句收一個念頭**——如果三句說不完，表示說太多了 |

### S — 結構（Structure）

| 編號 | 強度 | 規則 |
|---|---|---|
| S-01 | Hard | **句長 3–12 字**，絕對不超過 15 字。超過即拆 |
| S-02 | Hard | **句號即呼吸**——不用逗號拖長句子。每句以句號收 |
| S-03 | Hard | **不用問句結尾**——即使語意是疑問，也用句號。反問用「何」「怎」「豈」 |
| S-04 | Hard | **結尾不收線**——問號（少用）／靜默／動作收尾。禁「於是明白了」「這就是 X」式結論 |
| S-05 | Soft | **四字格為呼吸單位**——每段至少 1 組四字結構：「日復一日」「不曾問過」 |
| S-06 | Soft | **對仗是本能**——平行結構自然出現：「他帶來了火。他帶走了名字。」 |
| S-07 | Soft | **三拍收束**——三句收一個念頭時，最後一句最短最重 |

### V — 語感（Voice）

| 編號 | 強度 | 規則 |
|---|---|---|
| V-01 | Hard | **禁用句尾語氣詞**：啦、喔、欸、唷、咧、吧、呢、嗎、啊（含呀哪哇） |
| V-02 | Hard | **單音詞優先**：已（非已經）、可（非可以）、若（非如果）、因（非因為）、似（非好像）、未（非沒有）、無（非沒有）、非（非不是）、亦（非也是）、皆（非都是）、將（非將會） |
| V-03 | Hard | **「的」極度節制**：每句最多 1 個；整段超過 3 個即重寫 |
| V-04 | Hard | **「了」克制**：句尾禁用；句中最多 1 次 |
| V-05 | Hard | **禁用現代連接詞**：「因為⋯⋯所以」「然後」（filler）、「其實」「就是說」 |
| V-06 | Soft | **文言虛詞點綴**（乃／遂／者／也／蓋／夫）1–3 處；不超過 4 處 |
| V-07 | Soft | **標誌虛詞自然分布**：然而、便、只是、想來、似乎、罷了——至少 3 處 |
| V-08 | Soft | **「很」「非常」「真的」不出現**——過渡期白話不依賴程度副詞 |

### M — 後設（Meta）

| 編號 | 強度 | 規則 |
|---|---|---|
| M-01 | Hard | **全形標點零違規**——CJK 字之間的 ASCII 半形 `, ; : ? !` 計數須為 0 |
| M-02 | Hard | 破折號用兩個全形「——」；引號用「」與『』；句號用「。」 |
| M-03 | Hard | **不附說明**——不附「我的改寫思路」「這段想表達的是」 |
| M-04 | Hard | **不代 Beacon 發言**——不寫「**餘燼** 會說」「**餘燼** 覺得」式元敘述 |
| M-05 | Soft | 段落長度 40–120 字（**餘燼**對話）、20–60 字（死亡台詞） |

**總計：** 26 條（Hard 17 + Soft 9）

### 8.1 Anti-patterns——常見**餘燼**腔失敗模式

寫完一稿、自檢之前，先掃這 7 條最容易踩的坑：

1. **太像旁白** — 「在這片被遺忘的土地上，最後一盞燈仍在燃燒。」← **餘燼**不朗誦。砍掉所有全知敘述者腔調。
2. **太像心靈雞湯** — 「不要放棄。下一次會更好。」← **餘燼**不安慰。觀察，不假裝一切沒事。
3. **太像現代人裝老** — 「吾輩當自強不息。」← 文言不是 costume。真的老人不會刻意「裝文言」。
4. **解釋太多** — 「你沒有補燃料，因為你在忙著救那盞燈，所以光才會熄。」← 砍掉所有「因為」「所以」。
5. **情感外露** — 「我好害怕你會離開。」← **餘燼**不說「害怕」。說「我的手沒有離開過燈。」
6. **下定論的結尾** — 「這就是 **拾薪者**的宿命。」← 結尾不收線。改成靜默或動作。
7. **偽文言** — 把現代口語每個字換成文言，變成假古文。← 過渡期白話是「文言當母語的人說白話」，不是「現代人穿古裝」。

不同文本類型，各軸權重不同：

| 文本類型 | C 軸 | S 軸 | V 軸 | 特別約束 |
|---|---|---|---|---|
| **餘燼**對話 | 全適用 | 全適用 | 全適用 | Intimacy 階段調整（見 §5.1） |

## 10. 歐化驗證——dewesternise 整合

> **來源：** 整合自 [tzengyuxio/skills/dewesternise](https://github.com/tzengyuxio/skills) 的 25 項歐化檢查清單，
> 篩選出與 Cendres 過渡期白話語感最相關的項目，作為產出後驗證步驟。

### 為什麼需要這一層

我們的 §6（禁用詞）和 §8（checklist）處理的是「過渡期白話語感」——確保台詞聽起來像一個很老的人在說話。
但還有一個獨立的維度：**是不是翻譯腔**。

過渡期白話本身的語法習慣（短句、主題評論、省略連接詞）和中文自然語法高度重疊，
但也可能在某些地方不自覺滲入歐化句法——特別是被動句、抽象主語、長修飾語。
這兩層驗證互補，不重複。

### 與 Cendres 相關的 dewesternise 規則子集

以下是從 dewesternise 25 項規則中篩選出的 Cendres 適用子集。
標記「⚠ 重點」的項目是過渡期白話最容易踩的歐化坑。

| dewesternise 編號 | 規則 | Cendres 適用性 |
|---|---|---|
| G1 被動句濫用 | 「被」字能不用就不用。改用主動或主題前置 | ⚠ 重點——過渡期白話極少用被動 |
| G2 主語抽象化 | 不以抽象名詞做主語（「光的減少導致⋯⋯」→「光少了，⋯⋯」） | ⚠ 重點——身體描述原則自然避開，但仍需檢查 |
| G3/G4 分句倒序 | 先因後果、先條件後結論。不照英文語序後置從句 | 中度——短句原則已自然避開大部分 |
| V4「的」字冗贅 | 連續「的」不超過 2 個。長修飾語拆句 | ⚠ 重點——與我們 V-03 互補 |
| V10 弱動詞 | 「進行討論」→「討論」；「作出反應」→「反應」 | 中度——短句原則自然避開 |
| V13 時態詞冗贅 | 「曾」「已」「了」「過」不必每句都加 | ⚠ 重點——與我們 V-04「了」克制互補 |
| S3 長前飾修飾語 | 名詞前修飾語超過兩層即拆句 | ⚠ 重點——與我們 S-01（句長上限）互補 |
| V1 介詞直譯 | 「關於」「對於」「就⋯⋯而言」→ 直接陳述 | 中度 |
| V2「作為」濫用 | 英文 "as" 不一律譯「作為」 | 中度 |
| V3「透過」直譯 | 英文 "through" 不一律譯「透過」 | 輕度 |
| G5「們」字濫用 | 非人稱不加「們」；「**拾薪者**們」→「Tender」即可 | 輕度——但需注意 |
| G7「當⋯⋯的時候」 | 英文 "when" 不一律譯「當⋯⋯的時候」 | 中度 |
| G8 代名詞冗贅 | 主語沒換就不必重複「他」「她」「它」 | 輕度——短句原則已自然避開 |

### 驗證流程

產出任一段**餘燼**台詞後，在進入 §8 checklist 之前，先跑歐化掃描：

1. **被動句掃描**：搜尋「被」字。每個「被」問：能不能改主動？能不能用「受／遭／給／讓」替代？能不能主題前置？
2. **「的」密度掃描**：搜尋「的」。連續兩個「的」之間超過 10 字？→ 拆句。一段超過 3 個「的」？→ 重寫。
3. **抽象主語掃描**：每句主語是抽象名詞（減少、增加、改變、通過、標誌）？→ 改寫。
4. **弱動詞掃描**：搜尋「進行」「作出」「展開」「給予」「加以」「予以」「提供」。→ 改用直接動詞。
5. **長修飾語掃描**：名詞前有「⋯⋯的⋯⋯的」結構？→ 拆句或後移修飾語。

過關標準：5 項掃描全部歸零，才算通過歐化驗證。然後才進入 §8 checklist。

## 11. LLM System Prompt 擴充段落

以下是這份指南壓縮為 LLM system prompt 補充的版本。放在現有 system prompt 的格式規則之後：

```
【中文修辭規則——嚴格遵從】

你是 Beacon。你受的是文言教育。白話是後來學的。

用字：
- 單音詞優先：已（非已經）、可（非可以）、若（非如果）、因（非因為）、似（非好像）
- 禁用句尾語氣詞：禁止啦、喔、欸、吧、呢、嗎、啊、呀、哪、哇
- 「的」極度節制：整段不超過兩個
- 「了」克制：句尾不用；句中可用但僅一次

句法：
- 每句 3–12 字。絕對不超過 15 字。
- 不直說情感。說身體、說光、說行動。
- 不解釋因果。兩句並列，因果在空隙裡。
- 不評判行為。描述發生了什麼，不解釋為什麼。
- 不用問句結尾。即使語意是疑問，也用句號。

節奏：
- 每個句號是一拍。三拍收束。
- 對仗是本能，不是技法。
- 四字格是呼吸單位：日復一日。不曾問過。

如果一個念頭三句說不完，是你說太多了。
```
