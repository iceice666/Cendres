# 怎麼玩

---

## 設計備忘（開發者）

### 揭露哲學

Cendres 的資訊設計原則：**操作方法主動教，意義讓玩家自己發現。**

告訴玩家「按住互動鍵補燃料」是入場券——沒有這個他們沒辦法玩。告訴玩家「你在消耗的是什麼」是敘事——這需要時間、需要重複、需要他們自己準備好。前者缺席造成挫敗，後者太早出現破壞整個設計。

UI 提示原則：有操作映射的東西可以標示（按鍵、計時器、狀態視覺），沒有操作映射的東西不要標示（Tether 的第二種用法、見證動作、Garden 滿了的後果）。

### 教 vs. 不教

| 教（玩家指南涵蓋） | 不教（設計上的沉默） |
|---|---|
| 核心循環四階段 | 見證動作（Void 消散 1.5 秒視窗內按住互動） |
| Lumen 是資源，帶多帶少是決策 | Tether 聆聽模式（不拉取，獲得 Fragment） |
| 結構放置邏輯與補燃料操作 | Behemoth 進 Garden 的見證累積條件 |
| Lantern 四技能的操作與各類型取捨 | Void = 死者的真相 |
| Tether 捕捉的操作流程 | Garden 滿了後的無聲陰影（新捕捉的剪影） |
| Garden 餵食與燃燒／記憶兩條路的取捨 | Fragment 樹的深層分支（命名後的 Codex 深化） |
| 死亡後重置 vs. 永久保留的分界 | |

### 揭露時機備忘

這些機制**存在但不標示**——開發時確保 UI 不會意外提示它們：

- **見證動作**：Fragment 樹早期節點解鎖後說明「有個動作一直在那裡」。在那之前，Void 消散時的異常窗口頂多看起來像動畫特性。
- **Tether 聆聽模式**：Beacon 對話進度達一定深度，或 Codex 某物種達到第三層條目後解鎖說明。揭露前，長按 Tether 沒有拉取時 Void 的靜止只像是 timing 問題。
- **Behemoth 見證條件**：完全不說明。Run 15+ 自然出現，玩家如果選擇不攻擊，見證累積自己會發生。
- **Void = 死者**：敘事主線，透過 Beacon 對話和 Codex 深化慢慢揭露。永遠不要在教學、UI 或旁白裡點明。

## 玩家指南

### 你是誰

你是一個 Tender。你守著這座 Beacon——最後一個在 The Dimming 裡還亮著的火源。每晚，Void 從黑暗邊緣進來。你不是比它們強，你只是比它們更懂得用光。

今晚又要來了。

---

### Lumen：光即資源

Lumen 是遊戲裡唯一的資源，也是唯一的貨幣。

結構需要 Lumen 才能燃燒。你的 Lantern 需要 Lumen 才能出手。補燃料要 Lumen，捕捉要 Lumen。出發前，你決定帶多少走——帶少了，波次裡沒有彈性；帶多了，留給 Garden 和升級的就少了。

**Lumen 存量不會自動回復。** 消耗就是消耗，要去撿才有。波次結束後，被消滅的結構和 Void 會留下殘影，那是你的補給來源。

---

### Run 的節奏

每次 Run 分四個階段，循環重複直到你死或撐完所有波次。

**Reflection（出發前）**
你在 Beacon 旁。餵 Garden、花 Lumen 解鎖升級節點、選這次的 Lantern 類型和起始配置。這是唯一可以做長期決策的時間。

**Prep Phase（90 秒）**
第一人稱。在場地裡佈置光源結構、確認燃料儲量，聽聲音——第一批音效線索會透露即將到來的物種。時間有限，結構位置一旦波次開始就不能再移動。

**Wave Phase（不定長）**
Void 從黑暗邊緣進來。結構自動處理光源半徑內的目標，你負責補燃料、填補缺口、處理突破口，以及——如果你想——捕捉。

**Salvage Phase（60–90 秒）**
波次結束。在場地收集掉落的殘影，補充攜帶 Lumen。計時器可見——下次 Prep 馬上要開始。這個窗口在中後期會縮短。

---

### 結構：你的防線

光源結構是主要防線。它們在消耗 Lumen 的同時攻擊光源半徑內最近的 Void，自動運作，不需要你盯著。

**你和結構的三種互動：**

- **補燃料**：走近，按住互動鍵。消耗你攜帶的 Lumen，轉入結構儲量。這是波次中最頻繁的動作。
- **重新定位**：僅限 Prep Phase。波次一開始，位置鎖定。
- **主動拆除**：任何時候都可以。故意摧毀一個快撐不住的結構，立即回收 40% 燃料。在場地即將失守時，有時這比讓它被打爆更划算。

結構死亡兩種方式：燃料耗盡，或被 Void 接觸打壞耐久。兩者都讓它碎裂，掉出殘影。

**Lantern 類型（每次 Run 在 Reflection 選擇）**

| 類型 | 強項 | 代價 |
|---|---|---|
| Amber Core | Illuminate 半徑最大，視野覆蓋廣 | Flare 消耗大幅提高 |
| Ash Prism | Surge 持續時間長，橋接缺口更穩 | Illuminate 半徑減半 |
| Ember Wick | Flare 留下燃燒區域，持續傷害 | Tether 充能減少，捕捉機會變少 |
| Void-Tempered | Dim 效果強，Curious 氣質的 Void 幾乎不攻擊 | 基礎光源半徑最低 |

**Lantern 四個技能：**

- **Illuminate**：永遠激活。個人視野，持續消耗 Lumen。這是你能看見的原因。
- **Flare**：脈衝爆發。對半徑內所有 Void 造成傷害，高燃料消耗，用於緊急清場。
- **Surge**：延伸 Lantern 光橋接兩個結構之間的缺口，約 8 秒。讓穿越的 Void 仍在光源壓制內。
- **Dim**：將 Lantern 降至接近零輸出。Curious 氣質的 Void 靠近而非攻擊——但 Timid 氣質的 Void 會趁暗推進。Dim 是雙刃技能：先讀懂對面的氣質再用。代價：你幾乎失明。

---

### Tether：帶牠回來

Tether 是你的捕捉技能，也是唯一讓 Void 不死的選項。

**捕捉條件：**
1. 目標 HP 低於 25%（外緣發光變橙色）
2. 目標必須在任意光源半徑內
3. 瞄準並按住 Tether 輸入，維持 3 秒引導

**引導期間：**
- 你被定根（無法移動），但可以旋轉 Lantern 朝向任意方向
- Illuminate 光源維持，光半徑隨引導進度線性收縮
- 你自己的光 + 鄰近結構仍傷害進入範圍的 Void——旋轉可應對側翼威脅

**主動取消：** 放開 Tether 輸入。Void 保持當前 HP，不觸發 Feral。乾淨取消，沒有懲罰。

捕捉成功：Void 從場地移除，儲存。你獲得小量 Lumen 退款——比殺死它少，但它活著。

被中斷（受到攻擊超過閾值）：Void 恢復至 40% HP，暫時進入 Feral 狀態，攻擊性提高。

**張力所在：** 捕捉窗口往往是最危險的時機——低 HP 的 Void 旁邊常有健康的同伴。先清場、用 Surge 製造安全走廊，用 Dim 讓 Curious 氣質 Void 暫時不攻擊（注意 Timid 氣質在暗中反而更危險），或在引導中轉向應對進入泡泡的威脅，都是製造條件的方法。

Tether 有另一種用途。那個用途不在說明裡。

---

### Void Garden：燃燒還是記住

你捕捉的 Void 進入 Garden——Beacon Reflection 空間裡的一個角落。它們待在那裡，跨 run 永久保留。

**餵食：** 走近生物，按住互動鍵，給 Lumen。和補結構燃料是同一個動作。

Garden 裡的生物沒有數值狀態，只有視覺：毛色、光暈、動作幅度。缺乏照顧的生物會逐漸退化，行為變少，更像空的東西。它們不會死，只是看起來像是從來沒有人記得它們。

**兩條路，帶來的東西不一樣：**

| 路線 | 做法 | 回報 | 代價 |
|---|---|---|---|
| 燃燒路線 | 大量餵食，不深入研究 | 生物被動產生少量 Lumen | 生物逐漸變空洞，Codex 停在表層 |
| 記憶路線 | 觀察、投入 Fragment | 波次前的行為線索、Beacon 對話提示 | 效率較低，需要時間和 Fragment |

名字是發現的，不是設定的。你不會在一開始就知道怎麼做這件事。

Garden 有容量上限，預設每物種最多 3 隻，總計 10 隻。滿了，新捕捉的就沒有位置。

### 死亡後留下什麼

死亡不是重置。

| 留下來（永久） | 重置 |
|---|---|
| Beacon Memory 升級節點 | 攜帶的 Lumen |
| Fragment 樹解鎖進度 | 結構佈置 |
| Void Garden 生物 | Tether 充能數量 |
| Void Codex 條目 | 波次進度 |
| 已揭露的 Lore 碎片 | |
| Beacon 對話進度（Run 數永遠遞增） | |

**死亡時的 Lumen：** 你攜帶的 Lumen 裡，30% 轉化為「grief residue」存入 Beacon，下次 Reflection 時可以收集。剩下的 70% 消失。

帶越多 Lumen 死，留下的越多。但帶越多 Lumen，死的代價也越重。
