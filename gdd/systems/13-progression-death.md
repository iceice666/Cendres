# 玩家進程與死亡敘事

## 9. 升級系統：燃燒 vs. 記憶

玩家有兩條平行的成長路線，對應「燒掉靈魂」和「記住靈魂」的選擇：

### **餘念**（燃燒路線）= 數值升級

花費 Lumen（靈魂）解鎖。你從死者那裡取走力量，讓自己更強。
- Lantern 數值（半徑、消耗、Flare 傷害）
- 結構數值（被動衰竭率、耐久、容量上限）
- Lumen 攜帶上限
- Tether 充能數量

### Fragment 樹（記憶路線）= 機制升級

用 Fragment（身份碎片，見 8.5 聆聽模式 / 8.6 見證動作）解鎖。死者教你新的事，改變你能做什麼，而不只是讓你做得更多。

| 解鎖階段 | 能力 | 敘事意義 |
|---|---|---|
| 早期 | Tether 聆聽模式可見（有 UI 提示） | 你學會可以不收割 |
| 早期 | 見證動作有 UI 提示 | 你學會可以在場 |
| 中期 | 為 Garden 生物命名（觸發深層 Codex 條目） | 名字是記憶的核心 |
| 中期 | 向**餘燼** 提問的選擇性對話觸發解鎖 | 死者教你如何追問 |
| 後期 | Lantern 安撫模式——讓 Void 短暫靜止而不傷害 | 光不只是武器 |
| 後期 | **餘念** 存放節點（存靈魂而非燒）| 你選擇讓它繼續存在 |

### 兩條路線的玩家體驗

| | 燃燒路線（**餘念** 優先） | 記憶路線（Fragment 優先） |
|---|---|---|
| 戰鬥感受 | 更強、更快、更高效 | 稍難，但有更多工具選項 |
| Garden | 比較薄，生物較少深層行為 | 豐富，Codex 有更多深層條目 |
| Run 29 感受 | 「我有力量，但和死者沒有關係」 | 結局 C 是真實的，不是隱藏獎勵 |

兩條路都能通往 Run 29。結局 C 對走記憶路線的玩家才有完整情感重量。

## 10. 玩家進程 Meta-Arc（跨 run）

| Run 數 | 機制狀態 | 敘事狀態 |
|---|---|---|
| 1–5 | 學習物種，燃料經濟差 | **餘燼** 正式、引導性 |
| 6–10 | Void Garden 第一個馴化物種 | **餘燼** 變溫暖，提及「以前」 |
| 11–16 | 完整結構工具組解鎖 | **餘燼** 開始迴避問題 |
| 17–22 | **餘念**樹中深度 | 第一個 imprint；Remnant 說話 |
| 23–28 | 接近最佳 build；Behemoth 可能出現 | **餘燼** 承認省略了什麼；但不說是什麼 |
| 29 | 匯聚點——真相可以完整拼湊 | Run 29 的**餘燼** 演講 |
| 30+ | 後真相遊玩——所有結局可達 | 玩家在自己的時間線上選擇結局 |

## 11. 死亡與敘事整合

死亡畫面：緩慢淡化為灰色。**餘燼**說一句話——永遠不同，從不以明顯的方式安慰你。

**餘燼**死亡台詞範例：

*(Run 2)* 「燃料先走。然後是光。然後是你。我看過這個順序很多次了。」

*(Run 7)* 「Lurker 從北邊的缺口來。你現在會記住這件事了。」

*(Run 14)* 「你試著捕捉那個有名字的。我理解。我以前也會試的。」

*(Run 21)* 「你接近某個東西了。我能從你還沒開口問的問題看出來。」

*(Run 28)* 「我現在記得你的名字了。從以前就記得。我應該早點告訴你的。」

*(Run 29 — 匯聚後)* 「……是的。你知道了。我很高興。我想我很高興。我等著能高興，等了很久了。」
