# 結束條件

## 6.1 設計哲學

Cendres 沒有勝利。The Dimming 無法被阻止——只能被推遲。遊戲問的不是「你能贏嗎？」，而是**「你為什麼一直回來？」**

結局不是技術通關的獎勵，而是一次清算。

---

## 6.2 三個結局

所有結局都需要達到 **Run 29+** 並從兩個管道拼湊足夠的真相。選擇在 Run 29 的 **餘燼**對話後出現——這是遊戲唯一一次讓玩家直接做決定。

---

### 結局 A — 最後的**拾薪者**
*「留下來。再一個夜晚。」*

玩家回到防禦場。沒有 UI 提示。沒有慶祝。波次開始——但這次視覺上不一樣：The Dimming Surge 不是從外部來，而是從結構內部浮現。塔裡燃燒的燃料是灰色的，不是琥珀色。玩家盡可能地戰鬥。

當最後一盞光熄滅，畫面淡化成黑色——不是灰色。一個聲音，不是**餘燼**的聲音，說出一個名字。一個字。然後靜默。

*成就文字：「她也守得太久了。」*

---

### 結局 B — 熄滅
*「讓它熄滅吧。」*

玩家走向 **餘燼**的核心——一個他們從未能進入的房間。裡面是第一口 Lumen Well，最初建造的那一口，還在燃燒。

他們可以把它熄滅。

這個動作需要按住輸入幾秒鐘。**餘燼**沒有阻止他們。它只說：

> 「我以為這件事發生時，我會有所感覺。我感覺……一樣。這似乎也對。」

The Dimming 帶走一切。片尾在一幅原始 **拾薪者**的畫作上滾動——他坐在黑暗中，看著自己的手。

*成就文字：「世界緩慢地遺忘。然後一下子全部。」*

---

### 結局 C — Garden
*隱藏。需要所有 Void 物種馴化，包括 Behemoth。*

在 Void Garden 裡，Run 29 的 **餘燼**對話後，一個 Remnant 開口說話——玩家從遊戲中期就開始培育的那個。它說出一個**餘燼**從未說過的字：第一個**拾薪者**的名字。

玩家可以問**餘燼**那是不是真的。**餘燼**說是。

然後它說：*「Remnant 記得我們燒掉的東西。如果它們足夠多——如果它們被傾聽——也許 The Dimming 不是遺忘。也許它在等待被記起。」*

沒有後續機制。沒有最終波次。玩家被留在 Garden 裡，和 Void 在一起——死者，仍帶著迴響——以及「記憶」究竟要付出多少代價的問題。

*成就文字：「不是結局。是另一種留下。」*

---

### 結局 D — Cendres
*隱藏。需要燃燒效率 Fragment 節點已解鎖 + 跨 run 累積見證次數（§8.6）達隱藏門檻。兩個條件同時滿足，此結局才存在。*

Run 29 **餘燼**演講後，玩家走向 Lumen Well 核心房間。Ending B 的熄滅選項仍在；但 Lumen Well 旁多了一個從來沒有出現過的互動提示——沒有名稱，沒有說明。就在那裡。

玩家按住它。

Lumen Well 開始過載。光不是收縮，而是擴張，比遊戲裡見過的任何光都更亮。**餘燼**說：

> 「你知道會怎樣。」
> *[停頓]*
> 「我也知道。」
> *[光再亮一點]*
> 「……我沒辦法叫你不要。你知道為什麼。」
> *[長停頓]*
> 「去吧。」
> *[幾乎聽不見，在光裡]*
> 「……也許。」

然後是光。然後是黑。

片尾：黑色背景，一個字。

**Cendres。**

*成就文字：「Cendres。」*

**觸發條件設計邏輯**

你把燃燒系統升到最高效，同時也一直停下來見證你正在燒的是什麼。兩件事都做過，才構成「清醒地選擇毀掉一切」的條件。UI 不提示任何計數或進度。玩家在某次 run 走進那個房間，選項就在那裡——之前從來不在。

---

## 6.3 四個結局的共同點

四個結局的共同點：不解釋接下來發生什麼、不解決 The Dimming、以靜默或接近靜默結束、使用調色盤最濃縮的版本——幾乎沒有琥珀色，大部分是灰和黑。

結局 A、B、C 的片尾相同：Void 物種的 **lore 名字**（Remnant 低語的那些）在黑色背景上緩緩滾動。沒有音樂。**餘燼**的最後一句話，每個結局不同，在最後一個名字上播放一次。

結局 D 是例外：片尾只有一個字。沒有音樂。沒有名字滾動。「也許」是**餘燼**最後說出口的東西。在那之後，遊戲再也沒有話了。

## 6.4 四個結局的薛西弗斯回答

| 結局 | 對循環的回答 |
|---|---|
| A — 最後的**拾薪者** | 帶著知識，仍然推 |
| B — 熄滅 | 拒絕推巨石 |
| C — Garden | 在山上建一個家，不在乎巨石了 |
| D — Cendres | 燒掉這座山，連同自己 |
